# Developers performance Analysis using Survival analysis

This repository contains a dataset and a Jupyter notebook that perform survival analysis on developer milestones.

Contents
- `DSC-517-RS-DevelopersMilestones-Dataset.csv` — dataset used for analysis
- `Okoth_Survival_analysis.ipynb` — analysis notebook
- `Summary of findings presentation.pdf` — final presentation

How to run
1. Clone the repository.
2. (Optional) Create and activate a Python virtual environment.
3. Open `Okoth_Survival_analysis.ipynb` in Jupyter Notebook or JupyterLab and run the cells.
